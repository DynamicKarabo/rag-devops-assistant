RAG DevOps Assistant — Architecture

Production RAG system for DevOps documentation — observability-first with evaluation in CI

Query Pipeline

  • • User sends question → FastAPI /query
  • • Embed query with all-MiniLM-L6-v2
  • • Hybrid search Qdrant (dense + sparse)
  • • Top-K chunks → NVIDIA Nemotron via OpenRouter
  • • Return answer + source citations

Observability

  • • Custom Prometheus metrics (latency, tokens, cost, sources)
  • • 13-panel Grafana dashboard (p50/p95/p99 latency)
  • • Qdrant health monitoring
  • • Error rate by type (401, 429, 503)
  • • Cost estimation per query

CI/CD + Evaluation

  • • GitHub Actions: lint → test → eval → build
  • • RAGAS eval on every PR (20 Q&A benchmark)
  • • Retrieval precision + semantic similarity gates
  • • GHCR multi-arch image push
  • • k3s deployment with health checks
